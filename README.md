# Kudos ⚡️

A high-performance team recognition and workforce analytics platform built with Next.js and Supabase.

## 🚀 Quick Start

1. **Install Dependencies**: `npm install`
2. **Setup Env**: Copy `.env.example` to `.env.local` and add your Supabase keys.
3. **Run Dev**: `npm run dev`
4. **Access**: `http://localhost:3000`

## 🏗️ Technology Stack

- **Framework**: Next.js 15 (App Router)
- **Database**: Supabase (Postgres)
- **Styling**: Tailwind CSS
- **Visuals**: Lucide Icons
- **Deployment**: Vercel
